Shiro VS Spring Security

Apache Shiro
简单灵活，可脱离Spring，粒度较粗；可以自己拓展，适合通过资源进行权限控制；
Spring Security
复杂笨重，不可脱离Spring，力度更细；适合做数据权限控制；
