RBT(红黑树)

  • 每个节点是红色或者黑色
  • 根节点是黑色
  • 每个叶节点是黑色的
  • 如果一个节点是红色的，则它的两个儿子都是黑色的
  • 对于每个节点，从该结点到其叶子节点构成的所有路径上的黑色结点个数相同

AVL(自平衡二叉查找树)

  • 首先是一棵二叉查找树

    某节点的左子树节点值仅包含小于该节点值

    某节点的右子树节点值仅包含大于该节点值

    左右子树每个也必须是二叉查找树

  • 带有平衡条件，每个结点的左右子树的高度之差的绝对值（平衡因子）最多为1

RBT和AVL
