SpringCloud学习

SpringCloud提供了快速构建分布式中常见模式的工具，包括配置管理、服务发现、断路器、智能路由、微代理、控制总线等。SpringCloud的构建是基于SpringBoot的实现。

应用架构的发展历程：
单体应用架构 –> 分布式架构 –>面向服务的SOA架构 –> 微服务架构
SOA架构个人理解是多个应用之间通过网络通信的架构，其应用程序通过网络协议提供服务，消费服务，不同业务提供不同的服务。(阿里的服务治理框架Dubbo)

SpringCloud包包含以下组件：
服务治理组件 Eureka + 客户端负载均衡组件 Ribbon + 声明式服务调用组件 Feign + API网关治理组件 Zuul + 分布式配置中心组件 Config + 消息总线组件Bus + 消息驱动组件 Stream + 分布式服务跟踪组件 Sleuth.

服务治理：Spring Cloud Eureka

负责微服务架构中的服务治理功能，即各个微服务实例的自动化注册与发现。
SpringCloud Eureka 是由 Netflix Eureka实现的，即包含了服务端组件也包含了客户端组件。
